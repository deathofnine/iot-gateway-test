## 1. RPC 功能测试

### 1.1 DHT11 温湿度读取测试

| 用例ID | TC_RPC_001 |
|--------|------------|
| 用例名称 | DHT11 温湿度读取 |
| 用例描述 | 验证 RPC 调用 dht11_read 方法能正确读取温湿度数据 |
| 前置条件 | 1. 开发板运行正常，RPC 服务已启动<br>2. RPC 地址和端口已在 config.yaml 中正确配置<br>3. DHT11 传感器已正确连接 |
| 测试步骤 | 1. 建立 RPC 连接<br>2. 调用 dht11_read 方法<br>3. 获取返回结果 |
| 预期结果 | 返回非空数据，温度值在 0-50°C 范围，湿度值在 0-100% 范围 |
| 测试类型 | 功能测试 |
| 优先级 | 高 |
